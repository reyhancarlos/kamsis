PBAST v2 Encryptor / Decryptor

Website Enkripsi & Dekripsi PBAST v2

Antarmuka statis yang sederhana, profesional, dan siap di-upload ke GitHub maupun dideploy ke Netlify. Website ini mengenkripsi plaintext menjadi ciphertext HEX dan mendekripsi ciphertext HEX kembali ke plaintext menggunakan algoritma PBAST v2.

Input & Output

Cara Pakai

1. Masukkan plaintext dan key, lalu klik Encrypt.
2. Masukkan ciphertext HEX dan key, lalu klik Decrypt.
3. Ciphertext ditampilkan dalam format HEX tanpa spasi agar rapi dan mudah dipakai lagi.

Ringkasan Algoritma

Prime-based: menggunakan deret bilangan prima sebagai komponen shift.
Avalanche state: setiap karakter memengaruhi state berikutnya.
Bidirectional shift: posisi ganjil menambah shift, posisi genap mengurangi shift.
Output profesional: hasil akhir dibalik lalu dikonversi ke HEX.

Contoh Cepat

Plaintext: Kamsis 1
Key: carlos
Ciphertext: 3565E128FA19F3B1
